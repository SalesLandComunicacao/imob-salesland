# Proposta — Imobiliária Personal Shopper

Proposta comercial estática (HTML puro) hospedada em **https://imob.salesland.com.br**.

## Stack
- HTML/CSS puro, sem build
- Hospedado na Vercel (projeto `imob-salesland`)
- Fontes: Outfit + DM Serif Display (Google Fonts)

## Estrutura
```
.
├── index.html      # Proposta inteira em arquivo único
├── logo.svg        # Logo Salesland
├── vercel.json     # Config Vercel (cleanUrls)
└── README.md
```

## Editar
Abrir `index.html` em qualquer editor. Para preview local:
```bash
open index.html
```

## Deploy
```bash
vercel --prod
```

A Vercel já tem o domínio `imob.salesland.com.br` apontado pra este projeto.

## Conteúdo da proposta
- Hero + KPIs (4 pilares · 4 camadas · 30 dias · 24/7)
- Posicionamento (4 pilares de Personal Shopper)
- Antes vs Depois (7 dimensões da operação)
- 4 Camadas de Filtro (Criativo → Smart Page → Agente IA → CRM)
- Agente de IA de Atendimento + chat simulado
- Cadência de Follow-up (D0 → D90)
- Pilha Tecnológica (CRM · WhatsApp · Mídia · BI)
- Investimento — Motor IA 1.0 (R$18k + R$4k/mês) e Motor Base (R$15k + R$3k/mês)
- Acordo lado a lado
- CTA WhatsApp
